# May I introduce you to Captain RAYmond Holt?

Captain RAYmond Holt is a renderer programmed mainly in cpp, by me and my fellow student Julian for the Computer Graphics lecture at University. It can handle various functionalities every renderer should have, like sphere, rectangle and mesh intersections, principled and dielectric bsdfs, texturing, envmapping and a pathtracer. The chair provided us with a framework called lightwave, which supported us developing the renderer.
